Các chức năng vẫn trong quá trình phát triển:
 - Ở Tab Home
 - Ở Tab Schedules
 - Ở Tab Notifications:
   + Chưa thông báo ra màn hình khóa.
 - Ở Tab Menu gồm có:
   + Sub tab Settings -> Notifications
   + Sub tab Settings -> Appearance
   + Sub tab Settings -> Language
   + Sub tab Settings -> About -> Version
   + Sub tab Profile -> Address (dữ liệu địa chỉ đã cũ)
